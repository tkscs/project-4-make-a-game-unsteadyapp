# Project-4-Make-a-game

1. Initialize this repository

2. Install `pygame`

Try this (in Terminal or Anaconda Prompt):

```
conda activate cs2526
conda install pygame
```

If that doesn't work, run this:

```
conda activate cs2526
pip install pygame
```
